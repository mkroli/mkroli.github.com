mkroli.github.com
  • websum
    websum calculates the md5 checksum of the content behind urls.
  • waitfile
    waitfile waits for files beeing deleted.
  • pam_groupmember
    PAM module that checks groupmemberships.
  • innet
    A small utility to check if host is in a specific net.
  • abook
    Abook is an addressbook program. This version adds the ability to import vCards.
  • pam_ldap
    The pam_ldap module provides LDAP authentication and authorization for PAM. This version adds an additional configuration option to compare the login name with the ldap uid. This is to make the authentication process case sensitive.
  • knockd_patch
    This patch enables the knockd configuration to have multiple "command" and "stop_command" lines.
  • icmptx
    ICMPTX provides a tunnel via ICMP protocol. This version changes the default behavior of aborting on "sendto()" failed to continuing.
  • RotatingCounter
    With RotatingCounter one can count events at the moment they occur. RotatingCounter can then calculate the sum of events over a given period. It will automatically expire too old events.
